Objectives:

Data streams are everywhere, from F1 racing over electricity network to social media feeds. Data Stream mining or Real-Time. Analytics relies on and develops new incremental algorithms that process streams under strict ressource limitations. This course focuses on, as well as extends the methods implemented in open source tools as MOA. Students will learn to how select and apply an appropriate method for a given data stream problem; they will learn how to design and implement such algorithms; and they will learn how to evaluate and compare different solutions.

 


Programme to be followed:

1. Introduction

2. Stream Algorithmics

3. Concept Drift

4. Classification

5. Ensemble Methods

6. Clustering

7. Frequent Pattern Methods
