Prerequisites:

The course is designed as an introduction to geospatial data processing with open-source Python libraries to students that do not need to possess programming skills (as the main Python stuctures and conditional will be taught as well).

However, it is highly recommended to possess intermediate statistical and mathematical skills.

The students can bring their own laptops (with Linux, MacOS or Windows operating systems) or work with the available computers found in the classroom.
Objectives:

The general objective of the 30-hours course entitled “Big Data and Geospatial Data Science: Python Programming for Geospatial Querying and Analysis” is the introduce the students to the application of open-source Python libraries for use in geospatial and geomatics applications.

It is expected that the enrolled students will become more interested in the use open-source technologies for Geomatics engineering, as they hold great importance in the current scientific and didactic landscape.

Furthermore, it will contribute to their in the context of internationalization strategies that are assumed by the participating universities.
Programme to be followed:
  1. Introduction to Big Data
  2. Python Programming Language
    1. Work environment: Anaconda and Jupyter Notebook
    2. Python Structures
  3. Scientific Programming with Python
    1. Libraries for data processing

a)      Homogeneous data: NumPy

b)      Heterogeneous data: pandas

    1. Libraries for data visualization:

a)      Matplotlib

b)      seaborn

  1. Geo-data science: Geospatial data processing and analysis with Python
    1. Spatial analysis: Theory
    2. GeoPandas, PySAL
  2. Course Assignment
Conclusions and References
Course exam:
A geo-data science project built in the last day using the technologies studied in this course (applied to data analyzed in the course or to data chosen by the student).
